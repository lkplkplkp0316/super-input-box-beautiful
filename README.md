# 超级输入框 - Edge 浏览器插件

一个强大的 Edge 浏览器插件，让你可以同时向多个 AI 网站发送问题，无需复制粘贴。

## 功能特点

- **统一输入**：在侧边栏输入一次，自动发送到所有 AI 网站
- **悬浮输入框**：全局悬浮输入框，随时可用
- **多平台支持**：支持 ChatGPT、Gemini、Claude、Phind 等
- **快速打开**：一键快速打开常用的 AI 网站
- **标签管理**：自动检测并管理活跃的 AI 标签页

## 支持的网站

- ChatGPT (chatgpt.com)
- Gemini (gemini.google.com)
- Claude (claude.ai)
- Phind (phind.com)

## 安装方法

### 方式一：开发者模式安装

1. 打开 Edge 浏览器
2. 访问 `edge://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载解压缩的扩展"
5. 选择本插件文件夹

### 生成图标

在安装之前，你需要生成图标文件：

1. 用浏览器打开 `icons/create_icons.html`
2. 右键点击生成的图标，选择"图片另存为"
3. 分别保存为 `icon16.png`、`icon48.png`、`icon128.png` 到 `icons` 文件夹

或者使用任何 16x16、48x48、128x128 的 PNG 图片作为图标。

## 使用方法

### 使用侧边栏

1. 点击浏览器工具栏中的插件图标
2. 在侧边栏中点击快速打开按钮，打开你想要的 AI 网站
3. 在统一输入框中输入你的问题
4. 点击"发送到全部"按钮
5. 所有活跃的 AI 标签页都会收到并发送你的问题

### 使用悬浮输入框

1. 在侧边栏中开启"显示悬浮输入框"开关
2. 悬浮输入框会出现在所有网页上
3. 你可以拖动悬浮框到任意位置
4. 在悬浮框中输入问题并点击发送
5. 问题会同时发送到所有活跃的 AI 标签页

### 快捷键

- 在输入框中按 `Ctrl + Enter` (Mac 上是 `Cmd + Enter`) 快速发送

## 技术实现

- 使用 Manifest V3 标准
- 使用 Side Panel API 实现侧边栏
- 使用 Content Scripts 注入到目标网页
- 使用 Message Passing 实现组件间通信
- 自动检测各网站的输入框选择器

## 常见问题

**Q: 为什么有些网站无法自动发送？**

A: 网站可能会更新其 HTML 结构，导致选择器失效。我们会定期更新以支持最新的网站结构。

**Q: 悬浮输入框会干扰我正常浏览吗？**

A: 不会，悬浮输入框可以最小化，只显示一个小按钮，需要时再展开。

**Q: 支持其他 AI 网站吗？**

A: 目前支持主流的几个 AI 网站，你可以在 `content.js` 中添加其他网站的支持。

## 隐私说明

- 本插件不收集任何用户数据
- 所有操作都在本地完成
- 不向任何第三方服务器发送数据

## 开发者

如果你想添加对新网站的支持，请编辑 `content.js` 文件：

1. 在 `detectSite()` 函数中添加网站检测逻辑
2. 创建新的发送函数，如 `sendToNewSite()`
3. 在 `sendMessage()` 函数中添加对应的 case
